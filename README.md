# RLT Raumzuordnung – Suche & PDF

Statische Web-App (läuft im Browser) zum Laden einer Excel-Raumliste und Filtern nach:
- Anlage (z. B. 1.3 / 2.6)
- Raum / Beschreibung (Freitext)
- Etage, Fassade
- Bereich (NW/SO)
- Luftart (Zuluft allgemein/gekühlt, Abluft belastet/unbelastet)

Export: Trefferliste als PDF.

## Deployment über GitHub Pages
1. Repository erstellen
2. `index.html` ins Repo-Root
3. Settings → Pages → Deploy from branch → `main` / root
4. Die Seite unter der GitHub-Pages-URL öffnen

## Datenschutz
Die Datei wird nur lokal im Browser verarbeitet.
