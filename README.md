# RLT Raumzuordnung – Suche & PDF (GitHub Pages)

## Nutzung
1. Seite öffnen (GitHub Pages)
2. Excel-Datei auswählen (.xlsx)
3. Nach Anlage / Raum / Luftart / Bereich / Fassade filtern
4. Trefferliste als PDF herunterladen

## GitHub Pages aktivieren
- Repository → Settings → Pages
- Source: Deploy from a branch
- Branch: main / root
- Speichern

Dann ist die Seite unter der GitHub-Pages-URL erreichbar.
